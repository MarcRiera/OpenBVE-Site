The ats.cfg file format

The ats.cfg file is a plain text file consisting of a single line: the relative path to the plugin that is to be used.

Example of an ats.cfg file:
▶

myplugin.dll

If you don’t include the ats.cfg file, the default plugin for trains is used which provides basic implementations of ATS and ATC. Please find more information about the default safety systems here, in the tutorials on the left, and here.
